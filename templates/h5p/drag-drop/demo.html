Drag and Drop - 3 Categories
Description

Drag and drop functionality for learners to pick up a card and drop it into a container bucket. Example and template is configured to three drops.

Configuration

  • Step 1: Settings. Modify the size of the task area if you need more room for additional cards.
  • Step 2: Task. This area allows you to modify the default location of each of the cards. You can add or modify the cards using either text or images as the cover element.
  • Step 3: Edit each card. When creating a new card, ensure that you select all drop zones as this allows the user to make mistakes or put it into any of the drop buckets. If you need to add additional context, you can add in a hyperlink and use a Quote block to create the hyperlink and set to open in a new window.
  • Step 4: Edit each buckets. Apply a label and select what the corrects cards are for the bucket.
