Accordion
Description

Reduce the amount of text presented to readers by using this responsive accordion. Readers decide which headlines to take a closer look at by expanding the title. Excellent for providing an overview with optional in-depth explanations.Â 

Features
  • Add title and text for each expandable item.
  • You can define whether you want to use H2, H3 or H4 tags for titles.
  • Rich formatting can be used for the expanded text.
